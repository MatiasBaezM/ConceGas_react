# ConceGas - Plataforma de Venta de Gas

Este proyecto es una aplicación web moderna para la venta y gestión de gas licuado, desarrollada con:

- **React** (v19)
- **TypeScript**
- **Vite**
- **Bootstrap**

## Características

- **Tienda**: Catálogo de productos, carrito de compras.
- **Autenticación**: Login y registro con validación de RUT chileno.
- **Panel Administrativo**: Gestión de pedidos, productos y perfiles.

## Scripts Disponibles

- `npm run dev`: Inicia el servidor de desarrollo.
- `npm run build`: Construye la aplicación para producción.
- `npm run lint`: Ejecuta el linter.
